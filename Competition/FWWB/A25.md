## Requirements

> 基于给定目标城市，城市可由参赛选手自己选择。通过网络、书籍等方法 整理收集目标城市的文旅数据，包括但不限于城市的历史、名人、景点、饮食 特色、热门店铺等信息，构建语料库，并基于大模型微调的相关技术方法，**实现**基于大模型的语料库问答系统。其中对于大模型的选择可以选取一些开源模型，如 GLM、modelscope 开源的GPT-3 中文版本、LLaMA、BLOOM 等。

😹
### 需求分解

1. 目标城市选择：西安
2. ***西安市文旅数据搜集(>3000 条)***
3. 具体功能设计
	+ 问答
		+ ***支持多轮问答，即需携带之前的问答***
		+ 流式输出
	+ 上传文件（添加至语料库）
	+ ***接入天气预测***
	+ 模型可选
	+ ***历史记录***
	+ 网址解析
	+ ...
1. 前端(VUE)
	+ UI 设计（参考 GPT 等网站）
	+ 前端业务开发
		+ VUE
		+ Tailwind
		+ ...
1. 后端(KOA)
	+ 语料库搭建（向量数据库 Milvus）
	+ LLM 接入
		+ ChatGLM
		+ 文心一言
		+ GPT
		+ 星火
		+ ...
	+ 数据库表设计 (MongoDB)
	+ 接口(Interface)设计
	+ ...
